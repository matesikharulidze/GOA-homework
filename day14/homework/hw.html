რა არის პროგრამირება?

პროგრამირება არის პროცესი, რომლის დროსაც იქმნება ალგორითმები და იწერება კოდი კომპიუტერული პროგრამების შესაქმნელად. პროგრამისტები სხვადასხვა ენებს იყენებენ, მაგალითად Python, JavaScript, C++, Java

პროგრამირების ძირითადი განხრები:

  • ვებ-დეველოპმენტი - ვებსაიტებისა და აპლიკაციების შექმნა (HTML, CSS, JavaScript, PHP)
  • მობილური დეველოპმენტი - მობილური აპლიკაციების შექმნა (Kotlin, Swift, Flutter)
  • სისტემური პროგრამირება - ოპერაციული სისტემებისა და სისტემური პროგრამების შექმნა (C, C++)
  • ხელოვნური ინტელექტი - AI და Machine Learning (Python, TensorFlow, PyTorch)
  • გეიმ დეველოპმენტი - თამაშების შექმნა (Unity, Unreal Engine, C#)
